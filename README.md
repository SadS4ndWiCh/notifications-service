# 📢 Micro Serviço de Notificação

## 📖 Descrição
Micro Serviço de notificação feito com NestJS realizado no evento do Ignite feito pela Rocketseat

## ⚙️ Instalação

```bash
$ npm install
```

## 🚀 Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 🛠️ Teste

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
